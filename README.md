## 🪟 What vnkey does

vnkey is a Vietnamese input method for Windows, macOS, and Linux.

It helps you type Vietnamese with a normal keyboard. You can use it for chat, email, school work, documents, and web forms. The engine is built in Rust, so it runs with low overhead and starts fast.
